# AGENTIC AI Project

This folder contains the agentic AI project and phase-based RAG prototypes.

## Contents

- `agentic_rag_text2sql.py` — Example script combining RAG and text-to-SQL concepts.
- `basic-agenai-tutorial.py` — Basic agent AI tutorial code.
- `openai-agents-fundamentals.py` — Foundational examples for OpenAI agent usage.
- `phase1/` — Prototype and real API agent examples.
- `phase2/fastapi-agentic-rag-Project/` — FastAPI-based RAG system with vector search.
- `phase3/production-rag/` — Production-ready RAG application variant.

## Recommended workflow

1. Start with the top-level scripts for general agent concepts.
2. Explore `phase1/` for early prototypes and API experiments.
3. Move to `phase2/fastapi-agentic-rag-Project/` for a full FastAPI + RAG app.
4. Compare with `phase3/production-rag/` for the production version.

## Notes

- The `phase2` and `phase3` subprojects are structured around FastAPI, LangChain, ChromaDB, and PostgreSQL.
- Use the subproject `README.md` files for setup and running instructions.
